# 财务 APP 需求文档（v1.2・终极完整版）

## 文档信息

- 文档名称：财务 APP 需求文档
- 版本号：**v1.2**
- 产品定位：**本地隐私优先・财商教育・资产负债管理工具**
- 核心原则：不连接银行、不自动扣款、不修改真实数据、纯手动确认、本地加密存储

------

# 1. 需求背景（不变）

当前经济环境下行，普通个人缺乏财务认知、财商基础薄弱，无法区分资产与负债，不懂现金流管理。本 APP 以《穷爸爸富爸爸》为理论框架，提供**账面财务管理、股票 / 基金持仓管理、负债管理、财务规划、情景沙盘、财商教育**一体化工具，不涉及真实资金交易，专注帮助用户建立健康财务思维。

# 2. 目标用户（不变）

- 18–45 岁普通个人、职场人、家庭用户
- 理财新手、股票 / 基金小额投资者
- 有房贷 / 车贷 / 网贷 / 亲友借款的用户
- 希望低成本建立财务体系的财商学习者

# 3. 核心痛点（新增补充）

- 无法区分资产、负债、现金流
- 股票 / 基金无法计算成本、加仓、盈亏
- 贷款计算复杂，手动算不准
- 亲友借款无凭证、无期限、无利息，难以记录
- 记账繁琐、易忘、易疲劳
- 无财务目标、无未来预测、无风险沙盘
- 数据不安全、换手机丢失、隐私担忧
- 缺乏专业、简单、可落地的财商指导

# 4. 需求价值（不变）

- 用户：3 分钟建立财务体系，自动算成本、贷款、现金流
- 产品：记账 → 持仓 → 负债 → 规划 → 教育 → 沙盘 全闭环
- 理念：以《穷爸爸富爸爸》资产负债逻辑实现财商提升

------

# 5. 功能需求（原 v1.1 全部保留 + 本次 v1.2 新增）

## 5.1 账户管理（完整保留）

支持多账户：现金、微信、支付宝、储蓄卡、社保卡等

字段：名称、类型、余额、备注

操作：增删改查、余额调整、流水记录

**余额调整场景**：修正错误、现金赠与、资产盘盈等

**流水规则**：所有操作自动生成流水，形成闭环

## 5.2 资产管理（股票 / 基金 100% 完整保留）

### 5.2.1 通用资产

工资、租金、利息、版权、副业、被动收入、自定义

### 5.2.2 股票资产（完整保留）

首次买入：价格、股数、手续费 → 自动算成本价

多次买入：加权平均成本

卖出：价格、股数、手续费 → 先进先出（FIFO）

实时价格可手动修改，买卖自动同步实时价

展示：成本价、实时价、市值、浮动盈亏、交易记录

### 5.2.3 基金资产（完整保留）

买入：单位净值、份额、手续费、锁定期标记

多次买入：加权平均成本

卖出：净值、份额、手续费 → 先进先出

锁定期由用户自行管理

展示：成本价、实时价、市值、浮动盈亏、交易记录

## 5.3 负债管理（完整保留）

12 类负债：房贷、车贷、信用卡、消费贷、装修贷、助学贷款、网贷、电商分期、租金分期、亲友借款、经营贷、其他负债

通用字段：名称、类型、本金、剩余本金、是否计息、利率、借款日期、还款方式（分期 / 随借随还）、还款日、期数、绑定账户、备注、凭证上传（可选）

核心规则：等额本息 / 等额本金、提前还款重算、助学贷款计息开关、亲友借款无凭证 / 无利息 / 无期限

## 5.4 资产 — 负债 — 账户联动（完整保留）

资产收益自动入账

负债还款自动扣款

股票 / 基金卖出资金自动入账

所有操作生成流水

## 5.5 财务健康评估（完整保留）

收支、现金流、负债收入比＞40% 预警

等级：优秀 / 良好 / 注意 / 风险

## 5.6 财务知识模块（完整保留）

书籍精华、科普、观点、低费率教育、基础术语解释

## 5.7 新手初始化模版（完整保留）

提供虚拟数据模版，降低上手门槛

## 5.8 净资产增长率仪表盘（核心指标）

公式：(期末净资产 − 期初净资产) / 期初净资产

展示：月度 / 年度、目标、进度、趋势图

## 5.9 财务目标规划（沙盘模式，不修改真实数据）

创建目标：金额、期限、类型

反向计算每月需投入金额

生成规划现金流 → 提醒 → 一键辅助录入 → 用户手动确认

**不自动执行、不自动改余额**

## 5.10 情景模拟・财务沙盘（独立副本）

复制真实数据 → 模拟情景：

- 失业 3 个月

- 利率上涨 1%

- 股市下跌 20%

- 每月多存 1000

- 提前还清负债

  

  输出未来 6/12/36 月预测报表

  

  不影响真实数据

## 5.11 周期性提醒・核对清单

房贷、信用卡、定投、账单核对提醒

提醒 → 一键打开录入 → 手动确认

**不自动生成流水**

## 5.12 投资隐性成本展示

记录：申购费率、赎回费率、管理费

展示：累计手续费、费率对收益的长期影响

## 5.13 隐私加密备份 / 迁移

用户主密码加密数据库

导出 / 导入加密数据包

本地优先、无云端、不上传

## 5.14 数据标准化字段预留

股票代码、基金代码、贷款机构

便于未来扩展，不增加当前成本

## 5.15 退款・冲正・内部转账（真实场景必备）

### 功能描述

- **内部转账**：账户 A → 账户 B，不影响总资产
- **退款**：消费退款、信用卡退款、商家退款，冲减支出
- **冲正（撤销）**：一键撤销任意记录，自动恢复资产 / 负债 / 账户余额

### 价值

解决 90% 用户 “录错、转错、退款” 的核心痛点

## 5.16 多维度预算体系

### 功能描述

- 月总预算、分类预算（餐饮、购物、娱乐等）
- 周期：周 / 月 / 自定义
- 剩余可花金额、超支提醒（温和不焦虑）

### 价值

真正帮助用户控制消费，而非只记录消费

## 5.17 应急金测算（财商核心功能）

### 功能描述

- 应急金 = 月固定支出 × 3~6 倍
- 自动判断是否充足
- 不足时给出每月需补充金额

### 价值

行业最刚需、最专业、最能建立信任的功能

## 5.18 负债最优还款策略（仅建议，不执行）

### 功能描述

- 雪崩法：优先还利率最高（总利息最省）
- 雪球法：优先还金额最小（心理正反馈强）
- 展示：预计还清时间、节省利息

### 价值

用户无需懂金融，一键获得最优方案

## 5.19 投资资产配置分析

### 功能描述

自动分类：

- 现金类

- 固收类（存款、债券）

- 权益类（股票、基金）

- 另类（黄金、房租等）

  

  展示配置比例，提示集中风险（如全仓股票）

## 5.20 快速记账・桌面小组件

### 功能描述

桌面一键打开 → 输入金额 → 选分类 → 保存

全程 2 秒，解决 “懒得记”

## 5.21 智能分类记忆

自动记录用户上次选择：分类、账户、标签

高频项目置顶，大幅降低操作成本

## 5.22 财务年度 / 季度体检报告

自动生成：

- 收支趋势
- 资产增长率
- 负债下降幅度
- 投资收益
- 优化建议

### 价值

让用户清晰看到 “进步”，大幅提升留存

## 5.23 行为财商提示（边用边学）

操作时轻量提示，不打扰：

- “此负债年化利率 18%，优先级最高”
- “基金持有＜7 天，手续费较高”
- “单一股票占比＞20%，风险偏高”
- “应急金不足，建议优先储备”

## 5.24 术语悬浮词典

点击任意专业词汇弹出解释：

成本价、净值、等额本息、LPR、锁定期、年化、净资产

## 5.25 应用锁・隐私保护

- 指纹 / 面容解锁
- 摇一摇隐藏金额（公共场所安全）
- 数据脱敏导出（可隐藏金额 / 名称）

## 5.26 数据回滚・本地快照

每天自动生成一次本地快照

支持回滚到：昨天 / 上周 / 上月

防止误删、误改

## 5.27 21 天财商习惯养成计划

每日轻任务：

- 记 1 笔账

- 看 1 个财商知识点

- 检查财务健康

  

  完成本地勋章，无上传、无社交

## 5.28 财务里程碑激励

- 首次净资产为正

- 应急金达标

- 负债下降 50%

- 定投坚持 3 个月

  

  正向强化，降低焦虑感

# 7. 非功能需求（含体积预估）

# ==============================

## 7.1 安装包大小预估

- **Android：18MB ~ 25MB**

- iOS：20MB ~ 30MB

  

  说明：纯本地工具，无大量图片、视频、SDK

## 7.2 本地数据库大小预估

- 基础使用：**500KB ~ 2MB**
- 3 年数据（含图片凭证）：**10MB ~ 50MB**
- 极端大量记录：**≤ 100MB**

## 7.3 备份文件大小预估

- 普通备份：**1MB ~ 5MB**
- 含凭证图片：**5MB ~ 40MB**

## 7.4 体积过大优化策略（重要）

1. 凭证图片压缩

   

   上传后自动压缩至 ≤ 200KB / 张

2. 备份自动清理策略

   - 保留最近 **10 个备份**
   - 超过自动删除最旧的
   - 用户可手动调整保留数量

   

3. 日志与快照限制

   

   快照只保留 

   最近 30 天

   

   自动清理冗余数据

## 7.5 性能要求

- 启动 ≤ 1 秒
- 计算 ≤ 2 秒
- 全功能离线可用
- 无后台自启、无网络请求

## 7.6 兼容性

Android 8.0+ /iOS 14.0+

深色模式、护眼模式、超大字体

# 8. 风险与边界（合规完整版）

# ==============================

- 仅为**账面财务管理工具**，不连接银行、不做真实交易
- 不提供投资建议、不荐股、不荐基金
- 不计算产品年化、不做收益预测、不评级金融产品
- 股票 / 基金价格由用户手动填写
- 贷款计算仅供参考，以金融机构合同为准
- 所有数据本地加密存储，不上传云端
- 情景沙盘、规划沙盘仅供教育参考
- 无贷款导流、无征信、无金融资质相关内容
